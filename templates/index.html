An app for creating webhooks with the #TeamTrees Charity

Automate your applications when someone donates.

API

You can make requests to the following API to get information about the current status of the campaign

Get Trees

GET /

This will return the current number of trees donated

Get Trees

GET /donations

This will return a list of donations

This takes an query parameter of top. If it's set to True, it will return the top donations rather than the most recent ones.

The result will be a JSON array of objects in the format

{
"name":"MrBeast",
"treeCount":100000,
"message":"I dare you to take this space from me :)"
}
